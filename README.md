# Similify AI

An NLP-based document similarity analyzer that compares two text documents using TF-IDF vectorization and cosine similarity.

## Features

- Text preprocessing
- TF-IDF vectorization
- Cosine similarity scoring
- Document comparison
- Flask-based web interface

## Tech Stack

- Python
- Flask
- Scikit-learn
- NLP
- HTML/CSS

## Working

1. User uploads/pastes two documents.
2. Text is preprocessed.
3. TF-IDF vectors are generated.
4. Cosine similarity is calculated.
5. Similarity percentage is displayed.
